About me

I escaped a legal career to pursue my passion for creating things. My life-long curiosity has taken me to wonderful places, from the Puget Sound to the Great Wall, and left me speaking in a handful of languages. Czech (Native), English (C2), French (B1-2), Mandarin (HSK 3), Russian (A2)

I was working as a Social Media Manager before shifting into UI design and front-end web development. Transitioning into a new career, I didn’t take the most straightforward path, which, on the other hand, helped me pick up some extra skills, like a better understanding of UX, graphic design, and service design.

I design user interfaces for apps and websites and turn them into readable code using semantic HTML, modern CSS (Flexbox, CSS Grid), and vanilla JavaScript. My workflow includes Figma for designing and prototyping, Bootstrap and Sass libraries, git & Github for version control and cooperation, Gulp and npm scripts for automation, and Browserstack for cross-browser testing.
